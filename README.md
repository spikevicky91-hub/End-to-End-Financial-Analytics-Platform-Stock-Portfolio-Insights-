# End-to-End-Financial-Analytics-Platform-Stock-Portfolio-Insights-
I built a financial analytics platform that ingests stock market data via APIs, processes it using Airflow and dbt, stores it in Snowflake, and generates portfolio-level insights with KPIs like volatility and returns.

#New Enhancements
1. 2000+ rows dataset
2. Advanced dbt transformations
3. KPIs: returns, moving averages, volatility
